Tutorial Completo
Aprenda a usar o GERENCION do zero ao avançado — guia para Supervisores de Vendas
📘
Este tutorial cobre todas as funcionalidades do sistema. Leia uma vez do início ao fim e depois use como consulta rápida. O GERENCION foi desenvolvido para transformar dados do Power BI em inteligência comercial prática.
1
O que é o GERENCION e como ele funciona
O GERENCION é um sistema de supervisão comercial que transforma os dados brutos do Power BI em diagnósticos, alertas e planos de ação inteligentes. Ele não substitui o BI — ele o amplifica e cruza todas as informações.

Lógica do sistema:
1. Você alimenta o sistema com os dados do BI (Vendas + Lupa + Churn)
2. O sistema cruza tudo automaticamente
3. Gera diagnósticos, alertas, narrativas e sugestões de ação
4. Você prepara a apresentação com seu plano
5. A diretoria vê uma apresentação profissional e coerente

Onde os dados ficam salvos? No próprio dispositivo (localStorage do navegador). Use sempre o mesmo dispositivo ou exporte/importe regularmente.

O sistema funciona offline? Sim, após o primeiro carregamento funciona sem internet.
💡 Quando você baixa uma versão nova do arquivo HTML, seus dados NÃO somem — eles ficam no browser. Mas sempre exporte antes de trocar de dispositivo.
2
Primeiro Acesso — Configure seu Perfil
Antes de usar qualquer função, configure seu perfil. Vá em ⚙ Perfil no menu lateral e preencha todos os campos:

Empresa: Nome da empresa (ex: Arcofoods)
Regional: Nome da sua regional (ex: Norte Fluminense)
Supervisor: Seu nome completo
Cargo: Seu cargo
Assinatura: Como quer assinar mensagens
Contexto da Regional: Características da sua região — sazonalidade, perfil dos clientes, principais desafios. Aparece automaticamente no Relatório e na Apresentação.

Após preencher tudo, clique em Salvar Perfil.
💡 O Contexto da Regional é o que diferencia o seu relatório. Quanto mais detalhado, mais inteligente fica o diagnóstico automático.
3
Lançar BI — Alimentando o Sistema (Página 1 do BI)
Acesse ⊕ Lançar BI no menu. O módulo tem 3 blocos de colar:

Bloco 1 — Regional (Página 1):
No Power BI, selecione e copie a linha NORTE FLUMINENSE da tabela. Cole no campo e clique em ⚡ Processar Regional. O sistema mostra um preview dos 13 campos detectados — confira se estão corretos.

Bloco 2 — Vendedores (Página 1):
Como cada vendedor é copiado individualmente, cole um de cada vez e clique em ➕ Adicionar Vendedor. O sistema acumula. Faça para os 9 vendedores. A lista mostra quem já foi adicionado e quem falta.

Bloco 3 — Churn (Página 7):
Cole as linhas dos vendedores da tabela de Churn (sem cabeçalho, sem linha Total). Clique em ⚡ Processar Churn. O sistema calcula o Churn da Regional automaticamente.

Ao final clique em 💾 Salvar BI Completo.
💡 Se algum vendedor não for detectado automaticamente, confira se o código (ex: 017) está no início do nome copiado.
4
Lupa — Alimentando o Sistema (BI Gestor)
Os módulos Lupa alimentam o sistema com dados do BI Gestor — faturamento anual, SSS, Top 50 e Indicadores. Acesse cada um pelo menu Lupa (Diretoria):

🔍 Lupa · Faturamento: Cole a linha da Regional e depois a tabela de Vendedores do BI Gestor. O sistema gera a frase pronta (NOTA) para você falar na apresentação.

🔍 Lupa · Top 50: Cole a tabela completa do Top 50. O sistema classifica cada cliente como Forte, Risco ou Alto Risco e permite registrar a ação de recuperação.

🔍 Lupa · SSS: Cole a tabela SSS Cadastrado. O sistema analisa o crescimento dos clientes existentes separado dos novos.

🔍 Lupa · Indicadores: Cole a tabela de Indicadores (SKU/PDV, Margem MC, Venda Média). Gera a frase pronta por vendedor.

Importante: Todos esses dados se cruzam automaticamente com o BI de Vendas na Apresentação e no Relatório.
💡 Quanto mais módulos você alimentar, mais completo e inteligente fica o diagnóstico. O ideal é alimentar todos antes de cada apresentação para a diretoria.
5
Preparar Apresentação — Sugestões + Seu Plano
Este é o módulo mais importante antes de uma apresentação para a diretoria. Acesse 🎯 Preparar Apresentação no menu.

Para cada vendedor você verá:

3 Dimensões de Status:
• MÊS — projeção atual (verde ≥100% / amarelo ≥90% / vermelho abaixo de 90%)
• ANO — % da meta YTD e crescimento vs 2025
• RETENÇÃO — churn rate (verde ≤9% / amarelo ≤15% / vermelho acima)

Sugestões do Sistema (em amarelo — só você vê):
O sistema analisa todos os dados cruzados e sugere ações específicas baseadas nos problemas reais de cada vendedor.

Seu Plano de Ação (campo de texto livre):
Leia as sugestões, use como referência, e escreva com suas palavras o que você vai fazer. Este texto aparecerá na apresentação para a diretoria — as sugestões do sistema não aparecem.

Clique em 💾 Salvar Plano para cada vendedor.
💡 Faça isso no dia anterior à apresentação. Chegue com os planos prontos — você saberá exatamente o que falar para cada vendedor.
6
Dashboard — Sua Bússola Diária
O Dashboard é a primeira tela que você deve verificar toda manhã. Mostra o estado atual da regional em segundos.

Cards principais:
🟢 Projeção Regional: % de quanto a equipe já faturou em relação à meta do mês.
🟢 Saldo Positivação: Clientes faltando para bater o objetivo.
🟢 % Devolução: Limite máximo: 0,65%.
🟢 Saúde da Regional: Índice geral de 0 a 100.

Diagnóstico Rápido: Quantos vendedores estão saudáveis, em atenção e em situação crítica.

Alertas Preditivos: O sistema identifica padrões de risco antes que virem problema.
💡 Verifique o Dashboard toda manhã antes de qualquer outra atividade.
7
Relatório Executivo — Diagnóstico Completo Cruzado
O Relatório Executivo cruza TODOS os dados alimentados no sistema. Acesse 📄 Relatório Executivo e clique em Gerar Relatório.

O relatório inclui:
• Panorama regional — mês atual + acumulado do ano lado a lado
• Top 50 — clientes em queda com destaque
• Diagnóstico individual de cada vendedor — MÊS + ANO + SSS + Indicadores + Top50 cruzados
• Análise Integrada — narrativa inteligente explicando a situação de cada um
• Plano de Ação — o que você escreveu no Preparar Apresentação
• Conclusão executiva

Botões:
🖨 Imprimir: Gera PDF direto
💬 WhatsApp/PDF: Abre para salvar como PDF e enviar
💡 Gere o relatório antes de toda reunião com a diretoria. Ele já chega com tudo cruzado e os planos prontos.
8
Apresentação — Slides para a Diretoria
A Apresentação transforma todos os dados em slides profissionais. Acesse 🎯 Apresentação e clique em ▶ Iniciar Apresentação.

Estrutura dos slides:
1. Capa com situação geral da regional
2. Regional — Mês Atual (projeção, premiação, churn, positivação)
3. Regional — Ano YTD (faturamento, crescimento, meta)
4. Top 50 — clientes em queda com ação imediata
5. Churn e Positivação — todos os vendedores
6. Ranking da Equipe por projeção
7 a 15. Um slide por vendedor — MÊS + ANO + SSS + Indicadores + Top50 + Análise + Plano
16. Diretrizes finais

Pressione F11 para tela cheia. Use as setas do teclado para navegar.
💡 Os planos de ação que você escreveu aparecem nos slides dos vendedores. A diretoria vê seus planos, não as sugestões do sistema.
9
Análise da Equipe — Diagnóstico Individual
Acesse ◉ Análise da Equipe para ver cada vendedor com diagnóstico automático e alertas.

Para cada vendedor aparecem:
• Indicadores do mês (PROJ%, FAT%, MARG%, POSIT, SALDO, %DEV)
• Tags informativos (SKU/CLI, Venda Média, Churn%, Taxa Positivação)
• Alertas automáticos com ação recomendada
• Botão + Plano para abrir o Plano de Ação com o nome já preenchido

Consulte esta tela antes de ir a campo para saber exatamente o que conversar com cada vendedor.
💡 Use os alertas desta tela como roteiro de campo. Você chega na conversa preparado com dados na mão.
10
Score dos Vendedores
O Score é uma nota de 0 a 100 para uso interno de acompanhamento — não é apresentado como "score" para a diretoria. Os pesos refletem os KPIs reais da Arcofoods:

🏆 Projeção: 40pts — maior peso, define premiação
👥 Positivação: 25pts
📉 Churn: 20pts (limite 9%)
📊 Margem: 10pts
↩️ Devolução: 5pts

Score ≥70 = saudável · 50-69 = atenção · abaixo de 50 = crítico

Use para priorizar visitas de campo — comece sempre pelo menor score.
💡 Na apresentação para a diretoria não use a palavra "score" — use as 3 dimensões (Mês / Ano / Retenção) que são mais claras e fáceis de explicar.
11
Positivação e Churn
Positivação — quantos clientes compraram no mês.
Taxa ≥70% = OK · 60-70% = atenção · abaixo = urgente

Churn — % de clientes que pararam de comprar.
≤9% = OK · 9-15% = atenção · acima de 15% = crítico

O churn é o maior risco estrutural da regional. Um vendedor pode estar bem na projeção do mês mas com churn alto — significa que está conquistando novos clientes de um lado e perdendo de outro. A carteira não se consolida.

Ação imediata para churn alto:
1. Identificar os clientes que saíram
2. Ligar pessoalmente e entender o motivo
3. Apresentar proposta de retorno
4. Registrar no Plano de Ação
💡 É muito mais barato recuperar um cliente que saiu do que conquistar um novo. Priorize sempre a retenção.
12
Acompanhamento de Campo e Plano de Ação
🚗 Campo: Registre cada visita — vendedor, data, cidade, cliente e o que foi combinado. O histórico mostra que você é um supervisor presente e organizado.

◆ Plano de Ação: Sempre que identificar um problema, crie um plano estruturado com:
• Problema identificado
• Estratégia / Solução
• Responsável
• Prazo
• Prioridade (Alta / Média / Baixa)
• Status (Em andamento / Concluído / Pendente)

Os planos ativos aparecem automaticamente no Relatório Executivo e na Apresentação.
💡 Qualquer supervisor vê o problema. Poucos têm o plano estruturado. O Plano de Ação é o que diferencia um supervisor reativo de um estratégico.
13
Negociação Comercial
O módulo de Negociação cria propostas comerciais profissionais para aprovação interna.

Como usar:
1. Preencha: cliente, vendedor, concorrente, total de SKUs e contexto
2. Clique em + Adicionar Produto para cada item
3. Para cada produto informe: preço tabela, preço vendedor e preço ideal
4. O sistema calcula diferença, % de investimento e total
5. Clique em ✨ Gerar Proposta
6. Clique em 📄 Abrir PDF para salvar e enviar
💡 Uma proposta bem estruturada tem muito mais chance de aprovação do que um pedido verbal. Mostre o impacto financeiro e a justificativa estratégica.
14
Aniversariantes e Relacionamento
Cadastre vendedores e clientes estratégicos com data de aniversário.

O sistema avisa quando alguém faz aniversário hoje e ordena os próximos por data.

Botão WhatsApp: Abre com mensagem já preenchida — é só enviar.
Botão ✨ Msg IA: Copia um prompt para o ChatGPT gerar uma mensagem única e personalizada.
Botão 📅 Calendário: Adiciona ao Google Calendar com recorrência anual.
💡 Uma mensagem de aniversário do supervisor cria um vínculo que nenhum número de BI cria. É o toque humano que diferencia um líder.
15
Exportar e Importar Dados
Os dados ficam no dispositivo. Para transferir ou fazer backup:

Exportar: Clique em Exportar Dados no final do menu. Um arquivo .json é baixado com a data no nome.

Importar: No dispositivo de destino, clique em Importar Dados e selecione o arquivo .json.

Quando exportar:
• Antes de atualizar o arquivo HTML
• Antes de trocar de dispositivo
• Toda sexta-feira como backup semanal

Importante: Se você abrir o arquivo HTML novo e os dados sumirem, é porque estão em outro perfil de browser. Importe o .json e tudo volta.
💡 Exporte toda sexta. Salve no Google Drive com a data no nome. Nunca perca mais de uma semana de trabalho.
