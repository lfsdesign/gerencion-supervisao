Tutorial Completo
Aprenda a usar o GERENCION do zero ao avançado — guia para Supervisores de Vendas
📘
Este tutorial cobre todas as funcionalidades do sistema. Leia uma vez do início ao fim e depois use como consulta rápida. O GERENCION foi desenvolvido para transformar dados do Power BI em inteligência comercial prática.
1
O que é o GERENCION e como ele funciona
O GERENCION é um sistema de supervisão comercial que transforma os dados brutos do Power BI em diagnósticos, alertas e planos de ação inteligentes. Ele não substitui o BI — ele o amplifica.

Lógica do sistema:
1. Você lança os dados do BI semanalmente
2. O sistema calcula automaticamente o Score de cada vendedor
3. Gera diagnósticos, alertas e recomendações
4. Você age com base nessas informações

Onde os dados ficam salvos? No próprio dispositivo (localStorage do navegador). Por isso, use sempre o mesmo dispositivo ou exporte/importe os dados regularmente.

O sistema funciona offline? Sim, após o primeiro carregamento funciona sem internet.
💡 O GERENCION é sua ferramenta secreta. Enquanto outros supervisores chegam nas reuniões com achismos, você chega com dados, diagnósticos e planos de ação prontos.
2
Primeiro Acesso — Configure seu Perfil
Antes de usar qualquer função, configure seu perfil. Vá em ⚙ Perfil no menu lateral e preencha todos os campos:

Empresa: Nome da empresa (ex: Arcofoods)
Regional: Nome da sua regional (ex: Norte Fluminense)
Supervisor: Seu nome completo
Cargo: Seu cargo (ex: Supervisor de Vendas)
Assinatura: Como quer assinar mensagens e relatórios. Use duas linhas:
   Seu Nome
   Supervisor de Vendas - Nome da Empresa
Contexto da Regional: Descreva as características da sua região — sazonalidade, perfil dos clientes, principais desafios. Esse texto aparece automaticamente no Relatório Executivo.

Após preencher tudo, clique em Salvar Perfil.
💡 O Contexto da Regional é o que diferencia o seu relatório. Quanto mais detalhado, mais inteligente e personalizado fica o diagnóstico automático.
3
Lançar BI — O Coração do Sistema
Toda semana, após a atualização do Power BI, acesse ⊕ Lançar BI e preencha os dados em 3 etapas:

Etapa 1 — Regional (Página 1 do BI):
Preencha os dados da linha da sua regional na mesma ordem dos campos:
% OBJ FAT · % FAT Real · % PROJ · OBJ % Margem · % Margem Real · OBJ Posit · Real Posit · Saldo Posit · DEV R$ · % DEV · SKU/CLI · Venda Média · Preço/KG

Etapa 2 — Vendedores (Página 1 do BI):
Para cada vendedor, preencha os mesmos campos na mesma ordem. Se um campo não aparecer para aquele vendedor no BI, deixe em branco — o sistema trata como zero.

Etapa 3 — Positivação e Churn (Páginas 6 e 7 do BI):
Para cada vendedor: Positivados · Não Positivados · Churn Rate %

Ao final, clique em 💾 Salvar BI Regional e Vendedores e depois em 💾 Salvar Positivação e Churn.
💡 Salve sempre os dois botões separadamente. O sistema guarda o BI anterior automaticamente para o Comparativo Mensal — por isso, ao lançar um novo BI, o anterior é preservado.
4
Dashboard — Sua Bússola Diária
O Dashboard é a primeira tela que você deve verificar toda manhã. Ele mostra o estado atual da regional em segundos.

Como ler os 4 cards principais:
🟢 Projeção Regional: % de quanto a equipe já faturou em relação à meta do mês. Meta = 100%. Sua premiação começa em 90%.
🟢 Saldo Positivação: Quantos clientes ainda precisam comprar para bater o objetivo. Número negativo = clientes faltando. Positivo = meta batida.
🟢 % Devolução: Percentual de devolução da regional. Limite máximo: 0,65%. Acima disso é alerta vermelho.
🟢 Saúde da Regional: Score geral de 0 a 100 calculado pelo sistema com base em todos os indicadores.

Como ler os 4 cards secundários:
💰 Sua Premiação Atual: Baseada na projeção da equipe. Abaixo de 90% = sem premiação. 90-99% = parcial. 100-109% = base. 110%+ = máxima.
📊 Venda Média Regional: Ticket médio por cliente. Indicador preditivo — mostra onde você chegará.
📦 SKU/Cliente Regional: Média de produtos por cliente. Baixo = oportunidade de crescimento não explorada.
📈 Margem Regional: % de margem atual. Meta: 17,5%. Mínimo aceitável: 15,5%.

Diagnóstico Rápido: Resumo visual de quantos vendedores estão saudáveis, em atenção e em situação crítica.

Alertas Preditivos: O sistema identifica padrões de risco antes que virem problema. Leia cada alerta e aja.
💡 Verifique o Dashboard toda manhã antes de qualquer outra atividade. Os Alertas Preditivos são seu diferencial — eles avisam o que vai acontecer antes que aconteça.
5
Saúde da Regional — O Índice Geral
A Saúde da Regional é um índice de 0 a 100 que resume a performance de toda a equipe em um único número.

Como interpretar:
🟢 70 a 100: Regional saudável. Manter o ritmo e monitorar.
🟡 50 a 69: Atenção. Identifique os vendedores em queda e acione planos de ação.
🔴 0 a 49: Situação crítica. Intervenção imediata em toda a equipe.

Score por Indicador: Mostra qual indicador está puxando o score para baixo — projeção, margem, positivação, devolução ou churn.

Distribuição da Equipe: Quantos vendedores estão em cada faixa.

Score Individual: Score de cada vendedor em um card visual. Use para identificar rapidamente quem precisa de atenção.
💡 Leve o score da regional para as reuniões com a diretoria. É um número único que resume tudo — muito mais poderoso do que uma planilha cheia de dados.
6
Score dos Vendedores — Quem Precisa de Atenção Agora
O Score é uma nota de 0 a 100 calculada com os pesos reais da empresa:

🏆 Projeção: 40 pontos (maior peso — define sua premiação)
  • 110%+ = 40pts   • 100-109% = 35pts   • 90-99% = 20pts   • 80-89% = 8pts   • Abaixo de 80% = 0pts

👥 Positivação: 25 pontos
  • Taxa ≥90% = 25pts   • ≥75% = 18pts   • ≥60% = 10pts   • ≥45% = 4pts

📉 Churn: 20 pontos (limite 9%)
  • ≤9% = 20pts   • ≤15% = 10pts   • ≤20% = 4pts   • Acima = 0pts

📊 Margem: 10 pontos
  • ≥17,5% = 10pts   • ≥15,5% = 7pts   • ≥14,9% = 3pts

↩️ Devolução: 5 pontos
  • ≤0,65% = 5pts   • ≤1,3% = 2pts   • Acima = 0pts

Como usar: O ranking é ordenado do maior para o menor score. Vendedores com score abaixo de 50 precisam de intervenção urgente. Entre 50 e 70, monitoramento próximo. Acima de 70, manutenção.
💡 Use o Score para priorizar visitas de campo. Comece sempre pelo vendedor com menor score — ele tem o maior impacto na sua premiação e na saúde da regional.
7
Análise da Equipe — Diagnóstico Individual Profundo
Esta tela mostra cada vendedor com todos os seus indicadores e um diagnóstico automático gerado pelo sistema.

Como ler cada card de vendedor:
• PROJ%: % de projeção — se está abaixo de 90%, impacta sua premiação
• FAT%: % do faturamento real sobre o objetivo
• MARG%: % de margem real — meta 17,5%, mínimo 15,5%
• POSIT: Clientes positivados / objetivo
• SALDO: Diferença entre positivados e objetivo
• %DEV: % de devolução — limite 0,65%

Tags informativos: SKU/CLI, Venda Média, Preço/KG, Positivados, Não Positivados, Churn%, Taxa de Positivação.

Alertas automáticos: O sistema identifica e exibe em vermelho (ALERTA) ou amarelo (ATENÇÃO) cada problema encontrado, com a ação recomendada.

Botão + Plano: Abre o Plano de Ação com o nome do vendedor já preenchido. Use sempre que identificar um problema.
💡 Consulte esta tela antes de ir a campo. Ela diz exatamente o que conversar com cada vendedor — você chega na conversa preparado, com dados na mão.
8
Positivação — Controle da Base de Clientes
Positivação é quantos clientes da carteira compraram no mês. É um dos 3 maiores KPIs avaliados pela diretoria.

Como ler:
• Positivados: Clientes que compraram no mês
• Não Positivados: Clientes que não compraram — esses precisam ser ativados
• Taxa: % da base ativa que comprou. Meta: acima de 70%

Situação por vendedor:
🔴 URGENTE = mais de 20 clientes não positivados
🟡 ATENÇÃO = 10 a 20 clientes não positivados
🟢 OK = menos de 10 clientes não positivados

O que fazer: Para cada vendedor com clientes não positivados, crie um Plano de Ação com rota de reativação — ligue para os clientes, entenda o motivo e apresente uma proposta.
💡 Clientes não positivados são oportunidades perdidas, não problemas. Cada cliente que volta a comprar melhora sua positivação, sua projeção e sua premiação.
9
Churn — Taxa de Perda de Clientes
Churn é o percentual de clientes que pararam de comprar. É um dos 3 maiores KPIs avaliados pela diretoria.

Limite máximo: 9%
🟢 Até 9% = OK — dentro do aceitável
🟡 9% a 15% = ATENÇÃO — monitorar e agir
🔴 Acima de 15% = CRÍTICO — clientes migrando para a concorrência

O que fazer com churn alto:
1. Identifique quais clientes saíram
2. Ligue para cada um e entenda o motivo
3. Apresente uma proposta de retorno
4. Crie um Plano de Ação com prazo e responsável

Churn + Projeção baixa = alerta máximo: Essa combinação indica risco de queda acelerada no próximo mês. O sistema alerta automaticamente quando isso acontece.
💡 É muito mais fácil e barato recuperar um cliente que saiu do que conquistar um novo. Priorize a retenção antes da prospecção.
10
Comparativo Mensal — A Evolução que Impressiona
O Comparativo mostra a evolução de cada vendedor do mês anterior para o mês atual — todos os indicadores lado a lado.

Como ler:
• Score Ant / Score Atual: Evolução do score — subiu ou caiu?
• Evolução Score: +pts em verde (melhorou) ou -pts em vermelho (piorou)
• Delta Proj: Quantos pontos percentuais a projeção subiu ou caiu

Importante: Este módulo só funciona após o segundo lançamento de BI. Quando você lança um novo BI, o sistema salva o anterior automaticamente como base de comparação.

Como usar nas reuniões: Leve o Comparativo para mostrar evolução com dados concretos. "Esse vendedor melhorou 8 pontos no score este mês" é muito mais poderoso do que uma opinião.
💡 O Comparativo é sua melhor ferramenta para reuniões com a diretoria. Mostra que você acompanha a evolução da equipe de forma sistemática e com dados reais.
11
Acompanhamento de Campo — Registre Tudo
Registre cada visita de campo — cliente visitado, cidade, data, vendedor acompanhado e o que foi visto, combinado e os próximos passos.

O que registrar em Observações:
• O que foi identificado na visita
• O que foi combinado com o vendedor e/ou cliente
• Oportunidades identificadas
• Próximos passos e prazo

Por que registrar? O histórico de campo mostra que você é um supervisor presente e organizado. Em reuniões com a diretoria, você consegue dizer exatamente quando foi a campo, com quem e o que foi feito.

Dica de rotina: Registre imediatamente após a visita, ainda no carro ou no local. Nunca deixe para depois — os detalhes se perdem.
💡 Um supervisor que registra tudo demonstra profissionalismo e organização. Isso diferencia você de qualquer outro supervisor da empresa.
12
Plano de Ação — O que Você Está Fazendo para Resolver
Sempre que identificar um problema — na Análise da Equipe, nos Alertas ou nas visitas de campo — crie um Plano de Ação.

Campos do Plano:
• Problema identificado: Descreva o problema de forma clara e objetiva
• Estratégia / Solução: Como você vai resolver
• Responsável: Quem vai executar (vendedor, você, equipe)
• Prazo: Data limite para resolução
• Prioridade: Alta, Média ou Baixa
• Status: Em andamento, Concluído ou Pendente
• Detalhamento: Descreva em detalhes as ações e o resultado esperado

Os planos ativos aparecem automaticamente no Relatório Executivo.

Botão + Plano na Análise da Equipe: Ao clicar, abre o formulário com o nome do vendedor já preenchido — agiliza muito o processo.
💡 Qualquer supervisor vê o problema. Poucos têm o plano estruturado para resolver. O Plano de Ação é o que transforma um supervisor reativo em um supervisor estratégico.
13
Clientes Estratégicos — Monitoramento Prioritário
Cadastre os clientes mais importantes da regional para acompanhamento próximo — aqueles que têm maior volume, maior potencial ou que estão em risco.

Status disponíveis:
🟢 Estratégico: Cliente de alto volume — prioridade máxima
🟢 Expansão: Cliente com potencial de crescimento
🔴 Em risco: Cliente com sinais de saída — ação urgente
🟡 Oportunidade: Cliente com potencial não explorado
⚪ Ativo: Cliente regular em acompanhamento

Como usar: Em cada cliente, registre o histórico, o vendedor responsável e as próximas ações planejadas. Revise semanalmente.
💡 Os clientes "Em risco" devem ser visitados pessoalmente pelo supervisor. Sua presença demonstra ao cliente a importância que ele tem para a empresa.
14
Aniversariantes — Relacionamento que Fideliza
Cadastre todos os vendedores da equipe e os clientes estratégicos com data de aniversário, telefone e e-mail.

O sistema faz automaticamente:
• Avisa quando alguém faz aniversário hoje
• Mostra quantos dias faltam para cada aniversário
• Ordena os próximos aniversários por data

Botão WhatsApp: Abre o WhatsApp com uma mensagem padrão com emojis já preenchida, com sua assinatura. É só enviar.

Botão ✨ Msg IA: Gera uma mensagem única e personalizada com emojis, diferente a cada clique — para vendedores (linguagem de equipe) ou clientes (linguagem de parceria). Muito mais impactante do que uma mensagem genérica.

Botão E-mail: Abre o app de e-mail com a mensagem já preenchida.

Tipos: Cadastre como Vendedor, Cliente ou Outro — o sistema adapta o tom da mensagem automaticamente.
💡 Uma mensagem de aniversário do supervisor cria um vínculo que nenhum número de BI cria. É o toque humano que diferencia um líder de um gerente.
15
Relatório Executivo — Sua Apresentação Profissional
O Relatório Executivo é o documento mais completo do sistema. Clique em Gerar Relatório e o sistema cria automaticamente um diagnóstico profissional com:

• Situação da Regional: Status geral com projeção média
• Sua Premiação Atual: Baseada na projeção da equipe
• Contexto Regional: Texto do seu perfil + sazonalidade
• Termômetro da Regional: Todos os indicadores com diagnóstico
• Diagnóstico Individual: Análise de cada vendedor com ações recomendadas
• Positivação e Churn: Análise da base de clientes
• Planos de Ação: Todos os planos ativos
• Diretrizes do Mês: O que fazer agora, em ordem de prioridade
• Conclusão Executiva: Síntese em linguagem profissional

Botões de ação:
🖨 Imprimir: Imprime o relatório diretamente
💬 WhatsApp / PDF: Abre o relatório completo e colorido para salvar como PDF e enviar
✉ E-mail / PDF: Mesmo que WhatsApp + botão de copiar texto
💡 Gere o relatório antes de toda reunião. Ele transforma dados brutos em linguagem executiva profissional — você chega preparado enquanto outros chegam improvisando.
16
Negociação Comercial — Propostas Profissionais
O módulo de Negociação permite criar propostas comerciais estruturadas para aprovação interna quando precisar de condições especiais de preço.

Como usar:
1. Preencha os dados da negociação: cliente, vendedor, concorrente, total de SKUs e contexto/justificativa
2. Clique em + Adicionar Produto para cada item em negociação
3. Para cada produto, informe: nome, quantidade, preço tabela, preço vendedor e preço ideal
4. O sistema calcula automaticamente a diferença unitária, % de investimento e valor total
5. Clique em ✨ Gerar com IA para criar o documento formatado
6. Clique em 📄 Gerar PDF para abrir e salvar como PDF

O que o sistema calcula:
• Diferença entre preço vendedor e preço ideal
• Percentual de investimento sobre o preço tabela
• Valor total do investimento por produto
• Resumo do investimento total da negociação
💡 Uma proposta de negociação bem estruturada tem muito mais chance de aprovação do que um pedido verbal. Mostre o impacto financeiro e a justificativa estratégica.
17
Apresentação — Slides para a Diretoria
O modo Apresentação transforma todos os dados do sistema em slides profissionais para reuniões com a diretoria ou apresentações em data show.

Como usar:
1. Lance o BI antes de abrir a apresentação
2. Clique em ▶ Iniciar Apresentação
3. Uma nova janela abre com os slides
4. Conecte no data show e pressione F11 para tela cheia
5. Navegue com os botões Anterior/Próximo ou as setas do teclado

O que está em cada slide:
• Slide 1: Capa executiva com situação da regional
• Slide 2: Termômetro com todos os indicadores
• Slide 3: Diagnóstico geral da equipe
• Slide 4: Ranking completo por score
• Slides 5+: Um slide por vendedor com diagnóstico individual completo
• Slide Positivação/Churn: Detalhe da base de clientes
• Slide Planos de Ação: O que está sendo feito
• Slide Final: Diretrizes e conclusão executiva

A apresentação é 100% responsiva — funciona em tablet, notebook e telão.
💡 Imagina você conectar o notebook no data show e apresentar slides profissionais com dados reais da equipe. Isso impressiona qualquer diretoria e separa você de qualquer outro supervisor.
18
Exportar e Importar Dados — Nunca Perca suas Informações
Os dados ficam salvos no dispositivo. Para transferir entre dispositivos ou fazer backup, use as funções de exportar e importar no final do menu lateral.

Exportar Dados:
1. Clique em Exportar Dados no menu lateral
2. Um arquivo .json é baixado automaticamente com a data no nome
3. Envie pelo WhatsApp, e-mail ou Google Drive para o outro dispositivo

Importar Dados:
1. No dispositivo de destino, clique em Importar Dados
2. Selecione o arquivo .json baixado anteriormente
3. O sistema recarrega com todos os dados importados

Importante: Exporte sempre antes de atualizar o sistema. Os dados não são afetados pela atualização do arquivo HTML, mas é uma boa prática manter backups regulares.
💡 Exporte os dados toda sexta-feira como backup semanal. Salve no Google Drive com a data no nome. Se algo der errado, você nunca perde mais de uma semana de trabalho.

20
Dúvidas Frequentes
Os dados somem quando atualizo o sistema?
Não. Os dados ficam no localStorage do navegador, separado do arquivo HTML. Mas exporte antes de qualquer atualização por precaução.

Posso usar em dispositivos diferentes?
Sim, mas os dados são independentes em cada dispositivo. Use Exportar/Importar para sincronizar.

O Comparativo Mensal não aparece. Por quê?
Ele só funciona após o segundo lançamento de BI. O primeiro lançamento cria a base, o segundo cria a comparação.

O relatório não gera. O que fazer?
Verifique se lançou o BI. O relatório precisa de dados para gerar. Se lançou e ainda não funciona, exporte os dados, atualize o sistema e importe novamente.

A apresentação não abre. O que fazer?
Permita popups no navegador para o site do GERENCION. A apresentação abre em uma nova janela.

O botão ✨ Msg IA não funciona. O que fazer?
Verifique sua conexão com a internet. A geração de mensagens por IA requer conexão ativa.

Posso instalar como aplicativo no celular/tablet?
Sim! Abra no Chrome, toque nos 3 pontos e escolha "Adicionar à tela inicial". O GERENCION vira um app com ícone próprio.
💡 Em caso de dúvida não resolvida aqui, consulte o suporte. O GERENCION é atualizado regularmente com novas funcionalidades.
